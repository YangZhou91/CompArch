### Assembler Part

####Usage

Step1 : open the the 'src' folder in the terminal

Step2 : Use 'make' command to build project

Step3 : 'java Main <filename>' 
		For example: 'java Main fib.asm'
		
Step4 : The binary machine code stores in the 'bin_result.txt'
